Common Workflow Language Viewer

This tool visualises and lists the details of a workflow with its inputs, outputs and steps and packages the files into a downloadable Research Object Bundle (zip file with metadata in a manifest), allowing it to be easily viewed and shared.

This tool is currently experimental and may not work for all valid workflows.
